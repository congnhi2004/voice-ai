# Acceptance Checklist

Use this file as release evidence. Each item should be marked with status and concrete proof before launch.

Status values: `Not started`, `In progress`, `Pass`, `Fail`, `Blocked`.

Current release stance as of 2026-05-10: **public prototype pass with accepted risks; commercial production blocked**. The public runtime proves short OpenAI-backed TTS and video localization, local Docker/tmux runtime, local storage, and internal MLflow tracking. It does not prove Cloud Run, GCS durability, production Stripe billing, production auth hardening, Cloud Monitoring alerts, or async worker execution.
